## Image Optimization Tips

1. **Compression**: Use tools like TinyPNG or ImageOptim
2. **WebP Format**: Consider using WebP for better compression
3. **Responsive Images**: Provide multiple sizes for different devices
4. **Alt Text**: Always include descriptive alt text for accessibility
5. **Loading**: Use lazy loading for better performance

## File Naming Convention

All images follow the pattern: `[property-type]-[description].jpg`

Examples:
- `apartment-modern-2bhk.jpg`
- `villa-luxury-3bhk.jpg`
- `plot-residential-corner.jpg`
- `commercial-office-modern.jpg`
